# Elegant Fashion Shop

A responsive e-commerce website for a fashion shop specializing in Shoes, Ready-to-Wear Clothing, and Custom Tailoring Services.

## Features

- **Responsive Design**: Built with Bootstrap 5 for perfect responsiveness across all devices
- **Product Catalog**: Filter products by category (Shoes, Clothing)
- **Shopping Cart**: Add/remove items, update quantities, and persist cart data using localStorage
- **Tailoring Services**: Booking form with client-side validation
- **Modern UI**: Clean, elegant design with a minimal color palette

## Technologies Used

- HTML5
- CSS3 with Bootstrap 5
- Vanilla JavaScript

## Pages

1. **Home Page** - Features hero section, product catalog, and tailoring services
2. **Product Detail Page** - Detailed view of individual products

## Key Functionality

### Product Filtering
- Filter products by category without page reload
- Responsive grid layout that adapts to different screen sizes

### Shopping Cart
- Add products to cart
- Update item quantities
- Remove items from cart
- Cart persistence using localStorage
- Visual feedback when adding items

### Tailoring Services
- Booking form with validation
- Service type selection
- Date picker for appointments

## Setup

1. Clone the repository
2. Open `index.html` in your browser
3. No build process required - pure HTML, CSS, and JavaScript

## Customization

- Modify `styles.css` to change colors, fonts, and spacing
- Update product data in `script.js` to add/modify products
- Extend functionality by adding new features in `script.js`

## Browser Support

- Modern browsers (Chrome, Firefox, Safari, Edge)
